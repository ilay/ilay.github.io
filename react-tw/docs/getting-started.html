新手教學

JSFiddle #

開始 Hack React 最簡單的方法是用下列 JSFiddle 的 Hello World：

  • React JSFiddle
  • React JSFiddle 不含 JSX

新手教學包 (Starter Kit) #

首先請下載新手教學包。

前往下載

在新手教學包的根目錄，建立一個含有下面程式碼的 helloworld.html。

<!DOCTYPE html>
<html>
  <head>
    <script src="build/react.js"></script>
    <script src="build/JSXTransformer.js"></script>
  </head>
  <body>
    <div id="example"></div>
    <script type="text/jsx">
      React.render(
        <h1>Hello, world!</h1>,
        document.getElementById('example')
      );
    </script>
  </body>
</html>

在 JavaScript 程式碼中嵌入 XML 格式的程式碼稱為 JSX；可以前往 JSX 語法 中學習更多 JSX 相關的知識。為了把 JSX 轉換成標準的 JavaScript，我們使用 <script type="text/jsx"> 標籤包覆著含有 JSX 的程式碼，然後導入 JSXTransformer.js 函式庫來實現在瀏覽器裡的程式碼轉換

文件分離 #

你的 React JSX 程式碼可以寫在另外的文件裡。建立下面的 src/helloworld.js。

React.render(
  <h1>Hello, world!</h1>,
  document.getElementById('example')
);

然後在 helloworld.html 引用該文件：

<script type="text/jsx" src="src/helloworld.js"></script>

離線轉換 #

首先安裝命令列工具（依賴 npm）：

npm install -g react-tools

然後把你的 src/helloworld.js 文件轉成標準的 JavaScript:

jsx --watch src/ build/

只要你修改了， build/helloworld.js 文件會自動產生。

React.render(
  React.createElement('h1', null, 'Hello, world!'),
  document.getElementById('example')
);

對照下面更新你的 HTML 程式碼

<!DOCTYPE html>
<html>
  <head>
    <title>Hello React!</title>
    <script src="build/react.js"></script>
    <!-- 不需要 JSXTransformer！ -->
  </head>
  <body>
    <div id="example"></div>
    <script src="build/helloworld.js"></script>
  </body>
</html>

想試試 CommonJS？ #

如果你想在 browserify，webpack 或者其他相容 CommonJS 的模組系統裡使用 React，只要使用 react npm 套件包 即可。而且，jsx 轉換工具可以很輕鬆的整合到大部份包裝系統裡（不僅僅是 CommonJS）。
